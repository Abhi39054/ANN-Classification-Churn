
# How to deploy the app on streamlit.

1. Login to streamlit using github (preffered) / email : https://share.streamlit.io/?utm_source=streamlit&utm_medium=referral&utm_campaign=main&utm_content=-ss-streamlit-io-topright


2. Click on Create app

3. Paste the public repo link from the browser url ( NOT the SSH URL that used for clone. )

4. select the Repo branch: main

5. Select the streamlit app file name: app.py

6. It will automatically give you a public url to view.

7. Click on Deploy to deploy it.


Note: Make sure that the project is following the same structure as this project. (like having app.py, and requirements.txt file)
